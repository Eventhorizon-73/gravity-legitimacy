# Gravity: Formal Legitimacy Logic

**Version 3 — Belnap-Enabled Legitimacy Framework**  
Conflict-aware • Audit-grade • Never silently collapses uncertainty

## What is Gravity?

Gravity is a formal legitimacy evaluation engine that uses **Belnap’s four-valued logic** (T, F, N, B) to make high-stakes decisions while preserving truth, conflict, and epistemic uncertainty.

Instead of forcing every decision into simple true/false, Gravity tells you exactly *why* something is approved, denied, unknown, or in conflict — with full replay certificates for perfect auditability.

## Core Features

- Canonical 4-valued legitimacy function (L4)
- Immutable event-stream semantics
- Witness sets and full replay certificates
- Clean external rendering contract: `T = APPROVE`, `F/N/B = STOP + why`
- Designed for governance, compliance, authorization, and policy systems
